# One Button Kingdom 👑🔘

20,000 people. 1 button. Pure chaos.

You tap. Everyone gets an event every 100 clicks. 
DRAGON, GOLD, TAX, SWAP, or RESET. Blame the last guy. 

Live demo: [one-button-kingdom.vercel.app](https://one-button-kingdom.vercel.app) 

![gif of button mashing](demo.gif) *replace with 5s screen record*

### How to play 
1.  Tap the button. 
2.  Don’t be the guy who hits 99. 
3.  Lockout griefers if you’re fast enough. 
4.  Rage in the comments somewhere else.

### Host your own in 5 mins 
1.  `Create Supabase project` -> Run the SQL in `/sql/schema.sql` 
2.  Paste URL + anon key in `index.html` 
3.  `vercel --prod` or any static host

That’s it. No login. No accounts. No tracking.

### Tech 
Vanilla JS + Supabase Realtime. Free tier handles 20k. 
1 HTML file because we’re not animals.
